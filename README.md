
### Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ npm run dev
```
### On windows 
You can run Run Server (Starts the program).bat

### Open in your browser
[http://localhost:3000/](http://localhost:3000/)
